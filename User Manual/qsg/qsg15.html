Auxiliary: Alternator


The Type Listbox includes general auxiliary types. Select Alternator from the list. The ID field is automatically set to the first three letters of the Type. The ID is required to link the auxiliary to the supply power input in the driving cycle (as shown later). It must not contain space or special characters. The length is not limited.

Use the button to select the predefined Alternator input file Demo_AuxDevice.vaux in DemoData. The input file format is described here.
