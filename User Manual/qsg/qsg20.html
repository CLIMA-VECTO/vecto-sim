Driving Cycle Format


The three cycles used in this example use the following format:
(comma-separated, dot = decimal-mark)

<s>
The column containing the Distance travelled [m]. When using time-based cycles this column is replaced by <t>  for time [s]. If both identifiers are missing the data will be interpreted as 1Hz. If both exist distance will be ignored.

<v>
Vehicle speed input is always required, obviously. Unit is [km/h]. The .vacc file (see next step) limits both positive and negative (braking) acceleration.

<grad>
The Road Gradient is optional. [%]

<stop>
The Stopping Time is only needed for distance-based cycles, i.e. if a <s> column is provided. [s]

<Aux_xxx>
These columns contain the Supply Power input for each auxiliary defined in the .vecto file where xxx matches the ID in the Aux Dialog. Note that Aux ID's are not case sensitive that's why <Aux_Demo> will link to the the Auxiliary DEMO.
