Define .vacc file


Finally, switch to the Driver tab of the VECTO Editor.

Load the Demo.vacc file located in the Demo Data\12t Demo\Declaration folder. This file limits acceleration and deceleration of the distance-based Target Speed Cycle. The format is described here.

Note: The .vacc file is always required. When using time-based cycles (e.g. from measurement data) you may want to replace this file with dummy values that are below the actual measured accelerations.

Engine Start/Stop is disabled for this vehicle.

For this example activate Overspeed with max. 5[km/h] overspeed and a minimum target speed of 70[km/h].

Look-Ahead Coasting should be predefined by default.

Click to save and close the file (after entering a name) and include it in the Job List of the Main Form.
