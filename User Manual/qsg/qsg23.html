Quick Start Guide

Check settings


You should leave all settings here as they are by default.

The Mode selector allows to switch between STANDARD and BATCH mode. In this example the STANDARD mode is required.

The Cycle Distance Correction ensures that the calculated distance matches the given driving cycle.

Use gears/rpm's has no effect here since neither gear- nor rpm-input is defined in any of the three driving cycles.

Write modal results enables output of the .vmod files.

The Shutdown system after last job option is designed for calculations with a high number of job and/or cycle files (mostly in BATCH mode) that may take several hours or days to process. As the name suggests VECTO will shutdown the system after all jobs are done (with 100s delay to allow cancellation)
