VECTO CSV Format

The following format applies to all CSV (Comma-separated values) Input Files used in VECTO:

List Separator: Comma ","
Decimal-Mark: Dot "."
Comments: "c" at the beginning of the comment line. Number and position of comment lines is not limited.

Input Files:
  • Job File (.vecto)
  • Vehicle File (.vveh)
  • Engine File (.veng)
  • Gearbox File (.vgbx)
  • Driving Cycle (.vdri)
  • Speed dependent Cross Wind Correction Input File (.vcdv)
  • Cross Wind Correction via Vair & Beta Input File (.vcdb)
  • Retarder Torque Loss Map
  • Full Load and Drag Torque (.vfld)
  • Fuel Consumption Map (.vmap)
  • Transmission Loss Map
  • Auxiliary (.vaux)
  • Max. acceleration and brake curves (.vacc)
  • Shift polygons (.vgbs)
  • Torque Converter Characteristics (.vtcc)

Output Files:
  • Modal Results (.vmod)
  • Summary Results (.vsum)
  • Results Overview (.vres) - no description yet
