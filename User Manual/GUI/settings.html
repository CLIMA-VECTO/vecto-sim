Main Form / Settings



Working Directory
The Working Directory can be accessed in the VECTO File Browser via the "Work-Dir" button. Further it is used as root directory for relative file paths in the job list and cycle list (BATCH Mode) of the Main Form. In input files you can point to this directory using the <WORKDIR> wildcard
Example: In "<WORKDIR>\VECTO\Job1.vecto" the <WORKDIR> part will be replaced by the actual Working Directory.


Logfile Size Limit [MB]
Whenever the logfile (LOG.txt in the application folder) reaches this size it will be restarted. One backup is always stored as LOG_backup.txt.

In- and Output in JSON format
Enables in- and output in JSON format. Old .vecto, .vveh, .veng and .vgbx input files are supported and can be converted to JSON using the respective editors.
File Open Command
This command will be used to open CSV Input Files like Driving Cycles (.vdri). See: Run command
Name: Name of the command as it will be shown in the menu when clicking the button (beside the option to open with GRAPHi).
Command: The actual command.

Example: If the command is excel and the file is C:\VECTO\cycle1.vdri then VECTO will run: excel "C:\VECTO\cycle1.vdri"


Air Density [kg/m3]
The Air Density is needed to calculate the air resistance together with the Drag Coefficient and the Cross Sectional Area (see Vehicle Editor).


Fuel Density [kg/l]
The Fuel Density is used to calculate the l/km results.


CO2-to-Fuel Ratio [-]
Mass ratio (kgCO2 / kgFC) used to calculate CO2 emissions.


Reset All Settings
Alle values in the Settings dialog and Options tab will be restored to default values.
